Incture Homepage Redesign (Assignment)
Overview

This project is a redesigned homepage for Incture, created as part of a front-end development assignment. The goal was to reimagine the existing website with a modern layout, improved structure, and interactive UI elements while keeping the core brand and services intact.

Objective

Redesign the homepage with a clean and professional user interface

Improve content organization and visual hierarchy

Implement interactive components using JavaScript

Ensure responsiveness across different screen sizes

Technologies Used

HTML5

CSS (Flexbox, responsive design)

JavaScript (DOM manipulation, carousel functionality)

Features

Hero Section with a clear value proposition and call-to-action

About Section describing Incture and its mission

Services / Portfolio Section highlighting key offerings

SAP Solutions Carousel with interactive buttons for the user to navigate

Industry Solutions Section showcasing domain expertise

Contact Call-to-Action for user engagement

Responsive Design for desktop and mobile devices

Carousel Implementation

A custom-built JavaScript carousel was implemented for the SAP Solutions section.
Features include:

Navigating slides using previous/next arrow buttons

Dynamic translation based on slide width

Active dots to show where the slide is currently

Smooth transition animations

Project Structure

Incture_website.html → The main HTML file

style.css → Styling and layout

script.js → Script to create Interactive functionality

Design Approach

The design focuses on:

Simplicity

Effective visual hierarchy

Modern UI patterns inspired by SaaS websites

Future Improvements

Add animations on scroll

Enhance accessibility (ARIA labels, keyboard navigation)

Integrate backend for contact form

Improve carousel with touch/swipe support
